# 🍽️ Bomato Restaurant Finder

A Zomato-inspired food ordering and restaurant discovery web application with a modern user interface.

## Features
- 🍴 Restaurant listing and search
- ❤️ Favorites system
- 🌙 Dark mode
- 🛒 Food ordering
- 💳 Payment flow
- 📦 Order tracking
- 📱 Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- React
- Node.js 

## Installation
1. Clone the repository
2. Install dependencies
3. Run the application
